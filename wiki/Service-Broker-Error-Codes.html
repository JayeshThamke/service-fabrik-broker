Service-Broker-Error-Codes

When some error occurs with broker during any lifecycle operation. Service Broker will send the description which contains several error codes based on the type of error. Such error codes are listed below:

Broker Errors

10001: This is default error code

Bosh Errors

20002: Bad format error. Ex- Task id is not in correct format 20003: Bosh is unavailable 20004: Deployment is not found 20009: Deployment already exists

Docker Errors

30003: Docker unavailable

CF

40001: Default CF error

Store

50001: Default backup/restore error

More Debugging

Further admin can see the actual error message from corresponding crd

kubectl get director <id> -o json | jq -r '.status.error'
